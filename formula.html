因数分解の公式

$$x^2 + (a+b)x + ab = (x+a)(x+b)$$ $$a^3 + b^3 = (a+b)(a^2 - ab + b^2)$$ $$a^3 - b^3 = (a-b)(a^2 + ab + b^2)$$

一次不等式の性質

$a < b \Rightarrow a + c < b + c$
両辺に同じ数を加えても不等号の向きは変わらない
$a < b, c > 0 \Rightarrow ac < bc$
正の数を掛けると向きは変わらない
$a < b, c < 0 \Rightarrow ac > bc$
負の数を掛けると向きが逆転！

集合と命題

集合の基本記号と表し方

$a \in A$ $a$ は集合 $A$ の要素である
$a \notin A$ $a$ は集合 $A$ の要素でない
$A \subset B$ $A$ は $B$ の部分集合（$A \subseteq B$）
$\emptyset$ 空集合（要素を1つも持たない集合）
$A \cap B$ 共通部分（積集合）：$A$ にも $B$ にも属す
$A \cup B$ 和集合：$A$ または $B$ に属す
$\overline{A}$ 補集合：全体集合 $U$ の中で $A$ に属さない要素全体
$n(A)$ 集合 $A$ の要素の個数

要素の個数の公式（加法定理）

$$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$

$A$ と $B$ の重なり分を1回引く。

$$n(\overline{A}) = n(U) - n(A)$$

全体から $A$ を引いた残りが補集合。

ベン図：2集合の関係

ド・モルガンの法則

$$\overline{A \cap B} = \overline{A} \cup \overline{B}$$

「$A$ かつ $B$」でない ＝「$A$ でない」または「$B$ でない」

$$\overline{A \cup B} = \overline{A} \cap \overline{B}$$

「$A$ または $B$」でない ＝「$A$ でない」かつ「$B$ でない」

命題と真偽・逆・裏・対偶

「$p$ ならば $q$」（$p \Rightarrow q$）に対して：

逆 $q \Rightarrow p$ 真偽は元と無関係
裏 $\overline{p} \Rightarrow \overline{q}$ 真偽は元と無関係
対偶 $\overline{q} \Rightarrow \overline{p}$ 元と同値 ✓
💡 対偶を使った証明
「$p \Rightarrow q$」を直接示しにくいとき、同値な対偶「$\overline{q} \Rightarrow \overline{p}$」を代わりに示す。

4つの命題の関係

左上「$p \Rightarrow q$」を基準に、横は「逆」、縦は「裏」、斜めは「対偶」の関係

真偽の関係まとめ

元の命題 逆 裏 対偶
真 不定 不定 真
偽 不定 不定 偽

「逆」と「裏」は互いに対偶の関係にあるため、真偽が一致する

📝 例
$n$ は整数とする。命題「$n$ は奇数 $\Rightarrow$ $n$ は4の倍数でない」の対偶は「$n$ は4の倍数 $\Rightarrow$ $n$ は偶数」であり、これらの命題はともに真である。

必要条件・十分条件・必要十分条件

$p \Rightarrow q$ が真のとき
$p$ は $q$ の 十分条件
（逆に）$q$ は $p$ の必要条件
$q \Rightarrow p$ が真のとき
$p$ は $q$ の 必要条件
（逆に）$q$ は $p$ の十分条件
両方向が真のとき
$p \Leftrightarrow q$　必要十分条件
$p$ と $q$ は同値
例：「$x = 2$」は「$x^2 = 4$」の十分条件（逆は $x = -2$ の反例あり）

正弦定理 (Sine Rule)

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$
  • check_circle外接円の半径 $R$ との関係を示す
  • check_circle2辺と対角、または2角と辺が既知の場合に有効

余弦定理 (Cosine Rule)

$$a^2 = b^2 + c^2 - 2bc\cos A$$
$$\cos A = \frac{b^2 + c^2 - a^2}{2bc}$$

💡 3辺既知または2辺と夾角が既知の場合に利用します。

三角比の相互関係

$$\sin^2\theta + \cos^2\theta = 1$$
$$\tan\theta = \frac{\sin\theta}{\cos\theta}$$
$$1 + \tan^2\theta = \frac{1}{\cos^2\theta}$$

データの分析

分散と標準偏差

$$s^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2$$
$$s = \sqrt{s^2} \quad \text{（標準偏差）}$$

$\bar{x}$: 平均値、$n$: データ数

相関係数の公式

$$r = \frac{s_{xy}}{s_x s_y}$$

$s_{xy}$: 共分散Covariance

$s_x, s_y$: 標準偏差Standard Deviation

$-1 \leq r \leq 1$範囲

相関係数 $r$ と散布図の見方

正の相関
$0 < r \leq 1$
相関なし
$r \approx 0$
負の相関
$-1 \leq r < 0$
